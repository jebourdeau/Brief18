# Banking Spring API

## Getting Started

1. Clone the repository
2. Configure PostgreSQL connection in `application.yml` (If you have docker, you can run `docker compose up -d db`)
3. Run: `./gradlew bootRun` (or `./mvnw spring-boot:run`)
